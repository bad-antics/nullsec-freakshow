# 🎲 nullsec-miasma
**File Entropy Analyzer — Detect Hidden Data, Weak Crypto & Packed Malware**

> Randomness tells secrets. Miasma reveals them.

Measures Shannon entropy and byte distribution to detect encrypted payloads, packed malware, weak crypto, and steganographic content.

## ⚡ Quick Start
```bash
miasma scan /path/to/file        # Analyze a single file
miasma scan /path/to/dir -r      # Scan directory recursively
miasma map /path/to/file         # Entropy map of a file
```
