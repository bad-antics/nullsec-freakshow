# 📕 nullsec-necronomicon
**The Book That Should Not Be Read — System Dark Assessment**

The master dark grimoire. Performs a comprehensive 5-chapter system assessment: Flesh (health), Blood (network), Bones (filesystem), Spirits (processes), and Seals (security). Generates a threat level verdict.

## ⚡ Quick Start
```bash
necronomicon ritual                    # Full dark ritual assessment
necronomicon ritual --json-output report.json  # Save as JSON
necronomicon chapter blood             # Read one chapter
necronomicon verdict                   # Quick threat assessment
```
