# 🐉 nullsec-chimera

**Binary Polyglot Structure Validator** — Part of the NullSec Freakshow Suite

Detects files that are structurally valid as multiple file formats simultaneously (polyglots). Unlike simple magic-byte matching, chimera validates actual format structure and identifies the construction technique used.

## Commands

| Command | Description |
|---------|-------------|
| `chimera scan <file>` | Analyze a file for chimera traits |
| `chimera dissect <file>` | Deep analysis of polyglot construction |
| `chimera hunt <dir>` | Hunt for chimeras in a directory tree |
