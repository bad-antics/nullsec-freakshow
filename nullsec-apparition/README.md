# 👤 nullsec-apparition

**Environment Variable Security Audit** — Part of the NullSec Freakshow Suite

Finds secrets, PATH hijack vectors, and dangerous variables lurking in your environment.

## Commands

| Command | Description |
|---------|-------------|
| `apparition manifest` | Full environment security scan |
| `apparition secrets` | Find leaked secrets in env vars |
| `apparition path` | Audit PATH for hijack vulnerabilities |
