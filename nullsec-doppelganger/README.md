# 👥 nullsec-doppelganger
**File Identity Crisis Detector — Unmask File Impostors**

Detects files masquerading as other types, double extensions, polyglot files, and finds identical-content twins hiding under different names.

## ⚡ Quick Start
```bash
doppelganger unmask suspicious.pdf.exe    # Reveal true file type
doppelganger scan /home/user/Downloads    # Find all impostors
doppelganger twins /var/www               # Find duplicate content
doppelganger polyglot weird.file          # Check for multiple identities
```
