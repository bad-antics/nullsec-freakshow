# 🦂 nullsec-manticore

**TLS/SSL Certificate Chain Analyzer** — Part of the NullSec Freakshow Suite

Three-headed beast: cert inspection, chain validation, cipher audit. Its sting finds your weakest TLS link.

## Commands

| Command | Description |
|---------|-------------|
| `manticore sting <host>` | Inspect a host's TLS certificate |
| `manticore ciphers <host>` | Audit supported cipher suites |
| `manticore sweep <host1> <host2>...` | Sweep multiple hosts |
