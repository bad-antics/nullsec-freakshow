# 📖 nullsec-grimoire
**The Dark Book of Password Arts — Occult Password Generator**

Generates occult-themed passwords (chaos, hex, rune, sigil, leet styles), dark passphrases, analyzes password strength, and creates full credential sets.

## ⚡ Quick Start
```bash
grimoire conjure --style rune --length 32    # Rune password
grimoire phrase --words 6                    # Dark passphrase
grimoire analyze 'MyP@ssw0rd!'              # Analyze strength
grimoire credentials --count 10              # Generate credential set
```
