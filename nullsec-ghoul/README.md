# 👹 nullsec-ghoul

**Shared Library Injection Detector** — Part of the NullSec Freakshow Suite

Finds .so parasites feeding on your processes. Detects LD_PRELOAD injection, suspicious library paths, and library hijacking.

## Commands

| Command | Description |
|---------|-------------|
| `ghoul feed` | Full library injection scan |
| `ghoul preload` | Detect LD_PRELOAD injections |
| `ghoul stalk <pid>` | Scan a specific process |
