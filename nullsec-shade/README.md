# 🌑 nullsec-shade

**File Permission Anomaly Hunter** — Part of the NullSec Freakshow Suite

Lurks in the darkness of file permissions — world-writable configs, orphaned files, capabilities, SGID binaries.

## Commands

| Command | Description |
|---------|-------------|
| `shade lurk` | Full permission anomaly scan |
| `shade writable <dir>` | Find world-writable files |
| `shade orphans` | Find orphaned files |
| `shade caps` | Audit file capabilities |
