# 🕯️ nullsec-seance
**Network Necromancy — Resurrect Dead Connections**

Resurrects dead TCP connections, explores DNS graveyards, reads ARP spirit tables, and communes with listening ports to identify what answers.

## ⚡ Quick Start
```bash
seance resurrect              # Find dead/dying connections
seance graveyard               # DNS tombstones
seance spirits                 # ARP table spirits
seance commune 192.168.1.1 22  # Talk to a port
seance autopsy                 # Full network autopsy
```
