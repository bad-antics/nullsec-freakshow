# 🦎 nullsec-cryptid
**Hidden API & Endpoint Hunter — Find What Shouldn't Be Found**

Hunts for undocumented endpoints, hidden API paths, and secrets lurking in source code, binaries, and config files.

## ⚡ Quick Start
```bash
cryptid hunt ./src                  # Hunt endpoints in source
cryptid binary /usr/bin/suspicious  # Hunt in binaries
cryptid secrets /home/user/project  # Find env secrets
```
