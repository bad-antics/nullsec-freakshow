# 💀 nullsec-lich

**Kernel Module & Rootkit Surface Scanner** — Part of the NullSec Freakshow Suite

Commands the undead kernel. Lists modules, detects hidden ones, checks taint flags, scans for rootkit indicators.

## Commands

| Command | Description |
|---------|-------------|
| `lich command` | Full kernel security scan |
| `lich modules` | List loaded kernel modules |
| `lich taint` | Check kernel taint flags |
