# 🐍 nullsec-basilisk

**DNS Resolver Security Audit** — Part of the NullSec Freakshow Suite

Petrifies insecure DNS configurations. Audits resolv.conf, nameserver health, DNSSEC, DNS-over-TLS, and tunneling risk.

## Commands

| Command | Description |
|---------|-------------|
| `basilisk gaze` | Full DNS security audit |
| `basilisk probe <ns>` | Probe a specific nameserver |
