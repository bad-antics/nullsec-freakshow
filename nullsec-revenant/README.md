# 🧟 nullsec-revenant
**Zombie Process Hunter — Hunt the Undead**

Hunts zombie processes, orphans adopted by init, and deep sleepers that haven't used CPU in ages. Full graveyard reporting.

## ⚡ Quick Start
```bash
revenant zombies                  # Hunt zombies
revenant orphans                  # Find orphaned processes
revenant sleepers --min-age 7200  # Find deep sleepers
revenant graveyard                # Full graveyard report
```
