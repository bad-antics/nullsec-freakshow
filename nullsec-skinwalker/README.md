# 🐺 nullsec-skinwalker

**Process Mimicry Detector — It Wears Your Process's Face**

Detects processes pretending to be other processes: name spoofing, path mismatches, deleted binaries still running, and impostor patterns.

## ⚡ Quick Start
```bash
skinwalker scan          # Hunt all skinwalkers
skinwalker hunt sshd     # Find everything claiming to be sshd
skinwalker autopsy 1234  # Deep inspect a process
```
