# 🔮 nullsec-ouija
**Summon Spirits From Deleted Files — File Carving & Recovery**

Recovers deleted files from disk images, memory dumps, and raw binaries using magic byte signatures. Each recovered file is a "spirit" summoned from the dead.

## ⚡ Quick Start
```bash
ouija summon disk.img --output ./recovered   # Carve files from disk image
ouija seance suspicious.bin                   # Scan for ghostly remnants
ouija tombstone file.bin 0x1000 --length 512  # Hex dump at offset
```
